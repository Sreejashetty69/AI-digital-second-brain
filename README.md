# 🧠 AI Digital Second Brain Assistant

AI Digital Second Brain is a personalized AI assistant built using **Python** and the **Groq LLM API**.
It acts as a personal conversational assistant that understands the user's profile and provides intelligent responses, along with fun interactive features such as games, quizzes, and interesting facts.

---

## 🚀 Features

* 🤖 Personalized AI conversation based on user profile
* 🧠 Context-aware responses using chat memory
* 🎮 Interactive mini games (number guessing game)
* ❓ Quick quiz feature
* 💡 Random interesting facts
* 💻 Simple command-line interface

---

## 🛠 Technologies Used

* **Python**
* **Groq LLM API**
* **Prompt Engineering**
* **Command Line Interface (CLI)**

---

## 📦 Installation

Clone the repository

git clone https://github.com/yourusername/ai-digital-second-brain.git

Move into the project folder

cd ai-digital-second-brain

Install required libraries

pip install -r requirements.txt

---

## ▶️ Running the Project

Run the assistant using:

python second_brain.py

The AI assistant will start in the terminal.

---

## 🎯 Available Commands

| Command | Function                           |
| ------- | ---------------------------------- |
| `/fact` | Displays a random interesting fact |
| `/quiz` | Starts a quiz question             |
| `/game` | Starts the number guessing game    |
| `exit`  | Closes the assistant               |
